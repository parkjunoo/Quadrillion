# Mobile Game Top 10 Race Research Notes

## Scope

- Period: 2016-2025, the latest ten complete calendar years as of 2026-06-28.
- Subject: worldwide annual top-grossing mobile games.
- Video metric: rank score, not revenue dollars. `#1 = 10`, `#10 = 1`.

There is no single audited worldwide official dataset for all mobile-game IAP, ads, web shops, third-party Android stores, and China Android distribution. Public app-intelligence sources also use different definitions. To avoid mixing incompatible dollar figures in one chart, the video uses annual Top 10 rank position as the normalized value.

## Primary Sources Used

- Sensor Tower, Top Mobile Games by Worldwide Revenue 2016.
- Sensor Tower, Top Apps and Publishers 2017.
- Sensor Tower, PUBG Mobile Tops Billion-Dollar Mobile Games 2020.
- Sensor Tower, Billion-Dollar Mobile Games 2021.
- Mobilegamer.biz via AppMagic, 2022 top grossing mobile games.
- Mobilegamer.biz via AppMagic, 2023 top grossing mobile games.
- Mobilegamer.biz via AppMagic, 2024 top grossing mobile games.
- Mobilegamer.biz via AppMagic, 2025 top grossing mobile games.

## Method Notes

- 2022-2025 use Mobilegamer/AppMagic ranked charts, which publish Top 10 amounts. AppMagic excludes Apple/Google platform fees, ad revenue, webshop income, and China fragmented Android revenue.
- 2020-2021 use Sensor Tower ranking and revenue callouts for billion-dollar titles, with lower Top 10 continuation cross-checked against public app-intelligence ranks.
- 2018-2019 are reconciled from public Sensor Tower/data.ai/AppMagic annual rank reporting because comparable full Top 10 dollar tables were not available in one public source.
- PUBG Mobile is treated as the combined PUBG Mobile / Game for Peace franchise where sources combine the China and global versions.
- Pokemon names are written without accent marks in the CSV to keep the codebase ASCII-only.

## Interpretation

The video answers: "Which games were in the annual global mobile-game grossing Top 10 over the last 10 years, and how did the rank positions change?"

It does not answer: "How many exact dollars did each game make on a comparable accounting basis?"
